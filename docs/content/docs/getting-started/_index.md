This section walks you through everything you need to get Rondo up and running. From wiring your Raspberry Pi to deploying the container with Ansible.

Follow the pages in order if you're setting up from scratch:
1. [Hardware Setup](./hardware-setup): What you need and how to wire the button (plus the optional LED) to your Pi's GPIO pins.
1. [Installation](./installation): How to configure and run the Ansible playbook that deploys Rondo to your Pi.
